Import comments from an Echo export file.

== Description ==

This plugin will import comments from a Echo export file. 

Please provide feedback and bug reports in the [Forums](http://wordpress.org/tags/echo-comments-importer?forum_id=10#postform)

== Installation ==

Upload the Echo Comments Importer plugin to your blog, Activate it, then navigate to Tools -> Imports from your WordPress admin panel.
